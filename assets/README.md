# Design Assets

This directory holds all non-code visual assets for Omara.

## Structure

- `palettes/` — Color palettes (base16, matugen, hyprland, terminal, etc.)
- `graphics/` — SVGs, logos, illustrations, icons
- `fonts/` — Font files and configuration
- `brand/` — Official Omara wordmarks, ASCII art, boot splash assets

## Guidelines

- Keep source files (SVG, .ase, .gpl, etc.) whenever possible
- Provide both light and dark variants where relevant
- Document the source of any third-party assets

Place new assets in the most specific subfolder. If something doesn't fit, open an issue.
